# redlistshift

The goal of this project is to quantify how often species move between IUCN red list categories, specifically quantifying how often species get downlisted from higher categories like endangered to lower categories like vulnerable. 

- [`scratch.R`](./scratch.R): A scratch file for testing and prototyping code snippets.
- `paper.qmd` (coming soon): The main Quarto document for the manuscript.
